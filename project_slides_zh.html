Final Project Presentation

跨学习率调度的 LLM 预训练损失曲线预测

基于 Multi-Power Law 仓库，复现 Tissue/MPL，并系统检验 residual correction、稳健性实验与 NCPL-style surrogate。

1. 问题背景与目标

我们想预测的不是一个点，而是一整条训练损失曲线

研究内容

给定已有 LLM 预训练 loss curves，拟合能够从学习率序列预测 loss trajectory 的模型。

核心动机

真实训练中常需要改变 schedule，例如从 cosine 切到 WSD/WSDLD；重新训练大模型成本很高。

项目边界

不重新训练 LLM，只在课程提供的曲线上做可复现的拟合、外推、诊断与比较。

目标：在严格 cosine-to-WSD/WSDLD 设置下，回答现有 scaling-law 方法能否可靠外推，以及 residual correction 是否真的带来改进。

Core Questions

五个核心问题

  • MPL 能否从 cosine schedule 外推到 WSD/WSDLD schedule？
  • Tissue 的 decay memory 是否能解释学习率衰减阶段的误差结构？
  • 在 base predictor 后面加入 residual correction 是否有帮助？
  • full ridge 的失败是 bug，还是 cosine-only residual learning 的过拟合诊断？
  • NCPL-style neural surrogate 在小规模课程数据上是否值得作为主方法？

Pipeline

Step 1读取 loss curve 与 learning-rate schedule。

Step 3在 cosine residual 上训练 full ridge / FSL-light。
Step 4评估 WSD、WSDLD 与辅助曲线。

总体指标

RMSE、MAE、PREDE、WorstE、R2、final relative error、AUC relative error。

阶段指标

稳健性指标

alpha sweep、MPL multi-start、decay minus stable RMSE。

4. 方法与结果对比：Direction A

FSL-light：用更少的 schedule functional 做残差修正

Full ridge

使用 10 个特征，包括 log_s1、s1_sq、多种 decay memory 和 t_norm。结果显示容易过拟合 cosine residual。

FSL-light

只保留 bias、log_tau、eta_norm、decay_conv、is_decay，强调 intrinsic time 与 decay convolution。

Main Comparison

FSL-light 没有全面胜出，但验证了过拟合诊断

方法 平均 RMSE 平均 final relative error 解释
Tissue0.0101010.004618最稳定的简单 baseline。
Tissue + full ridge0.0236230.006110RMSE 变差，说明 full ridge 过拟合。
Tissue + FSL-light0.0222990.004571final error 接近 Tissue，但整体 RMSE 仍变差。
MPL0.0105710.010235强 baseline，AUC error 最低。
MPL + full ridge0.0283720.007102final error 有改善，但整条曲线误差变大。

结论不能写成“我们的 correction 全面更好”。更准确的说法是：复杂 residual correction 在 cosine-only 设置下很容易不可迁移。

Direction B

稳健性实验解释了结果为什么不稳定

Alpha sweep

更强正则化能降低 FSL-light 过拟合。alpha=1 时，Tissue + FSL-light 平均 RMSE 降到 0.015387，MPL + FSL-light 降到 0.017510，但仍没有稳定超过 baseline。

MPL multi-start

25M、100M、400M 的 RMSE std 分别约为 0.003145、0.003253、0.003979，说明非凸拟合存在初值敏感性。

Stage-wise Analysis

只看 final error 容易误判，阶段误差更关键

MPL 的 decay 问题

主测试曲线中，MPL 的 decay RMSE 明显高于 stable RMSE，平均 ratio 约为 5.70。

Tissue 更均衡

Tissue 的 decay/stable ratio 约为 1.44，虽然模型简单，但阶段表现更稳。

Correction 的副作用

full ridge 和 FSL-light 可能把误差从 decay 段转移到 stable 段，导致整体 RMSE 不降反升。

Direction C

NCPL-style surrogate 是高风险 baseline，而不是主结论

方法

  • 输入：log_model_size、step_norm、log_step 与 FSL-light schedule features。
  • 模型：带 StandardScaler 的 MLPRegressor。
  • 目标：预测 log(loss)，再指数变换回 loss。

5. 分析与思考

实验结果支持一个更克制的结论

主要发现

  • Tissue 和 MPL baseline 已经很强，尤其是在小数据设置下。
  • full ridge 的失败不是代码错误，而是 residual learning 的过拟合信号。
  • FSL-light 更可解释，但训练信息不足时仍不能保证泛化。
  • stage-wise metrics 比单一 final error 更能解释模型行为。

进一步思考

  • 如果能加入更多 schedule，residual correction 可能更可靠。
  • 如果有更多模型规模和训练日志，NCPL-style surrogate 才更有意义。
  • 未来可以把 FSL-light 作为低复杂度先验，与更多数据结合。
最终贡献：一个可复现的跨 schedule evaluation pipeline，以及对 residual correction 在小数据设置下何时失效的系统诊断。

Summary

一句话总结

本项目把 Multi-Power Law loss prediction 扩展为完整的跨学习率调度评估平台：复现 Tissue/MPL，加入 FSL-light、稳健性实验和 NCPL-style surrogate，并说明在课程数据规模下，简单可解释 baseline 比复杂 residual 或神经 surrogate 更可靠。

可复现

所有指标和图表由脚本生成，结果保存在 results/metrics 与 results/figures。

可解释

通过 decay memory、intrinsic time 和 stage-wise error 分析 schedule 影响。

可扩展

后续可接入更多 schedule、更大训练日志和更强 surrogate 模型。
