Final Project Explanation

跨学习率调度的 LLM 预训练损失曲线预测

本项目基于 Multi-Power Law 原仓库，构建一个可复现的 loss-curve prediction 实验 pipeline： 用 cosine schedule 拟合预测器，再外推到 WSD / WSDLD，并通过 Tissue baseline、full ridge 诊断和 FSL-light residual correction 分析学习率衰减阶段的系统误差。

项目目标 方法设计 实验流程 结果解读 输出文件 复现方式

1. 项目目标

真实的大模型训练经常会改变学习率调度，例如从 cosine 转向 WSD 或 WSDLD。这个项目关注的问题是： 如果只在一种学习率 schedule 上观察到 loss curve，能否预测另一种 schedule 下的完整损失曲线。

项目不重新训练 LLM，而是使用仓库中已有的 25M、100M、400M 三个规模的 loss curve CSV， 比较不同预测方法在跨 schedule 外推时的稳定性、误差结构和可解释性。

输入数据 loss_curve_repo/csv_25、csv_100、csv_400 中的 step、lr、loss 曲线。
训练设置 主实验只用 cosine_24000.csv 拟合，然后测试 WSD、WSDLD 和辅助泛化曲线。
核心问题 学习率衰减阶段是否会让 cosine-trained predictor 出现系统误差，轻量 correction 是否更稳。

2. 方法设计

项目实现了三层方法：先有基线预测器，再加入残差修正，最后用 FSL-light 降低 full ridge 的过拟合风险。

方法 作用 解释
MPL 主要强基线 使用 Multi-Power Law 建模累计学习率和学习率变化带来的 loss drop。
Tissue Annealing Law 可解释 baseline 用累计学习率 S1 和 decay memory S2 描述学习率退火影响。
Full ridge correction 过拟合诊断 使用较多 schedule 特征学习残差；当它在 WSD/WSDLD 上失败时，不是 bug，而是 cosine-only residual learning 的过拟合信号。
FSL-light correction 方向 A 主线 只保留 bias、log_tau、eta_norm、decay_conv、is_decay 等少量 schedule functional。
FSL-light 的目标不是保证一定超过 MPL，而是用更少、更可解释的特征替代 naive feature stacking。 即使结果不能全面胜出，它也能形成可信的 negative finding：跨 schedule residual correction 不能简单堆特征。

3. 实验流程

Step 1 读取每个模型规模下的 loss curve 和 learning-rate schedule。
Step 2 在 cosine 曲线上拟合 Tissue 与 MPL base predictor。
Step 3 对 base predictor 的 residual 分别训练 full ridge 与 FSL-light correction。
Step 4 在 WSD、WSDLD 和辅助曲线上预测完整 loss curve。
Step 5 输出整体指标、stage-wise 指标、alpha sweep、prediction CSV 和报告图表。

阶段划分

误差不仅看整条曲线，还分为 warmup_or_start、post_warmup_stable、 decay 和 long_horizon。其中 stable 与 decay 的对比最能说明 WSD/WSDLD 的学习率衰减是否带来了额外预测难度。

4. 当前结果摘要

以下数值来自 results/metrics/cross_schedule_metrics.csv，只统计 WSD 与 WSDLD 两条主测试曲线， 对 25M、100M、400M 三个规模取平均。

方法 平均 RMSE 平均 final relative error 平均 AUC relative error 说明
Tissue 0.010101 0.004618 0.002255 简单、可解释，当前整体稳定。
Tissue + full ridge 0.023623 0.006110 0.006522 RMSE 变差，说明 full 特征可能过拟合 cosine residual。
Tissue + FSL-light 0.022299 0.004571 0.006112 比 full ridge 略稳，但整条曲线仍未超过 Tissue baseline。
MPL 0.010571 0.010235 0.001878 强 baseline，适合作为主对照。
MPL + full ridge 0.028372 0.007102 0.007580 final error 有改善，但 RMSE 明显变差。
MPL + FSL-light 0.028353 0.010768 0.006129 没有超过 MPL baseline，但提供了轻量 correction 的对照证据。
最终汇报不应写成“FSL-light 必然更好”。更稳妥的结论是：本项目系统检验了跨 schedule residual correction， 并发现课程数据规模下，full ridge 的失败是重要诊断，FSL-light 提供了更克制、更有文献动机的替代方案。

results/figures/*.png
学习率曲线、预测诊断、FSL-light 对比和消融图。

6. 复现方式

在项目根目录运行以下命令即可重新生成测试、指标和图表：

pip install -r requirements.txt
python -m pytest tests -q
python scripts/run_reproduction.py
python scripts/run_cross_schedule.py
python scripts/run_ablation.py
python scripts/make_figures.py

推荐阅读顺序

先看 src/splits.py 理解实验划分，再看 src/features.py 和 src/correction.py 理解 residual correction，最后看 scripts/run_cross_schedule.py、scripts/run_ablation.py 和 scripts/make_figures.py 理解完整 pipeline。
