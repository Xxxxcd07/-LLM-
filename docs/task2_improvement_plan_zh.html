Task 2 Improvement Plan

基于 Task 2 参考文献的损失曲线预测项目提升计划

目标不是重新设计一个复杂模型，而是在现有 Tissue / MPL 复现基础上，把 method development 从容易过拟合的 full ridge correction，提升为有文献动机、可解释、可验证的 FSL-light residual correction。

1. 当前项目状态

项目已经覆盖 Task 2 的主体要求：使用课程给定的 loss curves，复现 Tissue annealing law 和 Multi-Power Law，并完成 cosine LRS 训练到 WSD / WSDLD LRS 测试的 cross-schedule evaluation。 现有结果目录中已经包含 reproduction、cross-schedule、stage-wise、ablation 和 correction coefficient 等表格。

复现基线 Tissue 和 MPL 均已实现，并能在 25M、100M、400M 三个规模上运行。
主实验 使用 cosine_24000.csv 训练，在 WSD / WSDLD 上评估。
诊断指标 已有 RMSE、MAE、final relative error、AUC error 和 stage-wise metrics。
主要缺口 当前 full ridge correction 在平均 RMSE 上不稳定，容易被质疑为过拟合。
方法 WSD/WSDLD 平均 RMSE 平均 final relative error 解释
Tissue 0.010101 0.004618 简单、可解释，当前整体表现稳定。
Tissue + full ridge 0.023623 0.006110 RMSE 变差，说明 full 特征组合可能过拟合 cosine 训练曲线。
MPL 0.010571 0.010235 强 baseline，适合作为主要对照对象。
MPL + full ridge 0.028372 0.007102 final error 有改善，但整条曲线 RMSE 明显变差，说明 correction 不够稳健。

因此，最终汇报中不应声称 full ridge correction 全面提升；更好的叙事是： full ridge 暴露了 cosine-only residual learning 的过拟合风险，接下来需要一个更克制、更有理论动机的轻量修正方法。

2. 参考文献给出的提升线索

文献 对本项目的启发 对应提升点
Tissue et al., 2024 用 S1 表示学习率累计面积，用 S2 表示带动量的学习率退火面积。 保留可解释的 decay memory，但不要加入过多自由特征。
Luo et al., 2024/2025 MPL 用多段学习率下降量和饱和幂律函数刻画 decay correction；同时提醒拟合非凸且参数敏感。 增加多初值或 alpha sweep，报告稳定性，而不是只报告单次结果。
Hu et al., 2024 WSD 的核心是 stable phase 加 decay phase，decay 阶段会带来明显 loss drop。 强调 stage-wise error，尤其检查 stable-to-decay transition。
Li et al., 2025 Functional Scaling Law 强调 intrinsic time 和 convolutional schedule functional。 设计 FSL-light 特征：用优化推进时间和轻量 decay convolution 替代 full ridge 特征堆叠。
Zhang et al., 2026 NCPL 说明配置到性能的神经 surrogate 有潜力，但依赖大量训练日志。 作为反例提醒：课程数据较少，不宜使用端到端神经模型；可做轻量 residual correction。

3. Brainstorm 后的三个可选提升方向

方向 A：FSL-light residual correction

用少量、可解释、受 FSL 启发的 schedule functional 修正 Tissue/MPL 残差。

  • 优点：文献动机强，复杂度低，能解释当前 full ridge 失败。
  • 风险：不保证一定超过 MPL baseline，但即便失败也能形成可信 negative finding。
  • 结论价值：最适合作为 method development 主线。

方向 B：稳健性和不确定性实验

增加 MPL 多初值、ridge alpha sweep、stage-wise sensitivity analysis。

  • 优点：能增强实验严谨性，答辩时更好回应“结果是否稳定”。
  • 风险：方法创新度不如方向 A。
  • 结论价值：适合作为方向 A 的配套验证。

方向 C：NCPL-style 神经 surrogate

把模型规模、LRS、step 等作为配置输入，用 MLP 或树模型预测 loss。

  • 优点：和 Zhang et al. 对齐，形式灵活。
  • 风险：当前数据太少，容易过拟合，可解释性弱。
  • 结论价值：不建议作为本项目主线，最多作为未来工作。

推荐采用方向 A，并用方向 B 增强可信度。方向 C 保留在 slides 的 future work 中即可。

4. 推荐方法设计：FSL-light

FSL-light 的目标是替代当前过重的 full ridge correction。它仍然以 Tissue/MPL 为 base predictor， 只学习残差，但残差特征不再是大而全的列表，而是围绕 intrinsic time 和 decay convolution 设计。

Step 1 读取 loss curve 与 learning-rate schedule。
Step 2 使用 cosine 曲线拟合 Tissue/MPL base predictor。
Step 3 在 cosine residual 上拟合 FSL-light correction。
Step 4 在 WSD/WSDLD 上预测完整 loss curve。
Step 5 对比 baseline、full ridge、FSL-light 和 stage-wise error。

候选特征

  • bias：整体残差偏移，作为最小 correction。
  • log_tau：intrinsic time，使用 log(cumsum(lr)) 或归一化累计学习率表示优化推进程度。
  • eta_norm：当前学习率相对峰值，捕捉 stable 与 decay 阶段的 LR level 差异。
  • decay_conv：对正向学习率下降量做指数或 intrinsic-time convolution，借鉴 Tissue 的 S2 与 Li/FSL 的 schedule functional。
  • is_decay：是否已经进入 decay 相关阶段，仅作为低复杂度阶段指示。

关键约束：FSL-light 的特征数量应明显少于 full ridge，避免只在一条 cosine 训练曲线上学习出不可迁移的残差形状。

5. 实验计划

主实验

  1. 保持现有 strict split：只用 cosine_24000.csv 训练。
  2. 在 wsd_20000_24000.csv 和 wsdld_20000_24000.csv 上评估。
  3. 比较 tissue、mpl、full ridge、fsl_light。
  4. 报告 RMSE、final relative error、AUC relative error 和 stage-wise RMSE。

稳健性实验

  1. 加入 ridge alpha sweep：例如 1e-6、1e-4、1e-2、1。
  2. 如果时间允许，加入 MPL 多初值拟合，报告均值和标准差。
  3. 把 WSD/WSDLD 的 stable 与 decay 阶段分开比较。
  4. 保留 wsdcon、constant、cosine long-horizon 曲线作为辅助泛化检查。

新增输出

  • results/metrics/fsl_light_metrics.csv：FSL-light 主结果。
  • results/metrics/alpha_sweep_metrics.csv：alpha 敏感性结果。
  • results/figures/fsl_light_comparison.png：baseline、full ridge、FSL-light 的主对比图。
  • results/figures/stage_fsl_light_error.png：stable 与 decay 阶段误差图。

6. 成功标准与可能结论

情况 如何解释 答辩时的说法
FSL-light 优于 full ridge，接近或优于 baseline 说明文献动机约束能减少 cosine-only residual learning 的过拟合。 我们的贡献是用低复杂度 schedule functional 替代 naive feature stacking。
FSL-light 不能超过 MPL，但比 full ridge 稳 说明 MPL 已经很强，残差修正的空间有限，但轻量方法比 full ridge 更可靠。 这是一个有价值的 negative finding：跨 schedule residual correction 不能简单堆特征。
FSL-light 也不稳定 说明只用一条 cosine 曲线训练 correction 可能信息不足。 后续需要更多 schedule 或配置数据，这与 Zhang/NCPL 对大规模日志的依赖相呼应。

最重要的是不要把结论写成“我们的方法必然更好”。更稳妥、更成熟的表达是： 本项目系统检验了跨 schedule residual correction 的可行性，并发现轻量、文献约束的 correction 比 naive full ridge 更适合作为课程数据规模下的 method development。

8. 最终答辩叙事

推荐用下面的顺序讲：首先说明 cosine、WSD、multi-step 都是现实 LLM 训练中的重要 schedule； 然后展示 Tissue 和 MPL 的复现；接着指出 cosine-trained predictor 在 WSD decay 阶段仍有系统误差； 再展示 naive full ridge 反而过拟合；最后引出 FSL-light，说明我们参考 FSL 的 intrinsic-time 和 schedule functional，把 correction 做得更小、更可解释，并用 ablation 和 stage-wise metrics 验证。

这条叙事的优点是诚实、有层次，并且能把 reference review、experimental reproduction、method development 三个 Task 2 评分点连成一条线。
